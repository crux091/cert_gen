# Quick Start Guide

Get your certificate generator up and running in under 5 minutes!

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Bulk Generation

### Step 1: Prepare Excel File
Create a spreadsheet with names:
```
Name
Alice Johnson
Bob Smith
Carol White
```

Save as `.xlsx` or `.csv`

### Step 2: Upload and Export
1. Click "Upload Excel" in the sidebar
2. Select the element to replace (usually the name)
3. Click "Export Bulk"
4. Download the ZIP file with all certificates!

## Tips & Tricks

- **Lock elements** to prevent accidental changes
- **Save layouts** for reuse with "Save Current Layout"
- **Duplicate elements** with the copy icon
- **Adjust z-index** to layer elements correctly
- **Toggle dark mode** with the moon/sun icon
